## 📖 About The Project

This project is a high-fidelity clone of [PolicyBazaar](https://www.policybazaar.com). It was built as a collaborative effort by a dedicated team of 5 Full-Stack Web Developers, executed entirely within a one-week sprint. The application aims to provide a seamless user experience for browsing and purchasing varied insurance policies.

## ✨ Key Features

- **🏠 Home Page:** A dynamic landing page showcasing available insurance domains.
- **🔐 Authentication:** User Login & Sign-up functionality.
- **📝 User Details Form:** Interactive data collection tailored to user demographic records.
- **📊 Policy Details Page:** Detailed comparisons of insurance quotes, coverages, and premiums.
- **🛒 Checkout Page:** Organized invoice overview before payment.
- **💳 Payment Gateway:** A mock payment portal concluding the transaction seamlessly.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, React.js
- **State Management:** Redux
- **UI Components:** Chakra-UI, Material UI
